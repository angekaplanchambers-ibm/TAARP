# Designer Starter Kit

Learn the designer skill by working through exercises with a real project.

## Setup

1. Load the skill (read `SKILL.md`)
2. Bring your own project materials (see below)
3. Work through exercises in order

## Bring Your Own Project

This starter kit teaches design methodology through exercises. You'll need to supply your own reference materials:

- **A PRD or product spec** - requirements for the feature you're designing
- **A technical RFC or architecture doc** (optional) - system constraints and data model
- **Raw interview notes or research** (optional) - user research data to synthesize

Place your materials in a `reference/` folder alongside the exercises.

## Exercises

| # | Exercise | What You'll Do |
|---|----------|----------------|
| 1 | User Research | Plan research, synthesize notes, generate insights, create principles |
| 2 | Digest PRD | JTBD, OOUX, user journeys, UXDR, propose UI |
| 3 | UI Scan | Heuristic evaluation, pattern checks |
| 4 | Documentation | Design principles, specs, handoff docs |

## Getting Started

Load the designer skill, then open `exercises/01-user-research.md` and try the prompts.

## Tips

- Each exercise builds on the previous
- Save outputs as you go
- Raw research notes are intentionally messy - that's the point
- Adapt prompts to reference your own project materials
