# Exercise 3: UI Scan

Evaluate wireframes or existing UI against heuristics and patterns.

---

## 1. Heuristic Evaluation

Prompts to try:

- "evaluate this wireframe against Nielsen's heuristics"
- "what usability issues do you see?"
- "rate severity of each finding"
- "check this design against our design principles"

---

## 2. Accessibility Review

Prompts to try:

- "what accessibility concerns should I address?"
- "review this for keyboard navigation"
- "check color contrast and text sizing"

---

## 3. Pattern Check

Prompts to try:

- "is this consistent with existing patterns in the product?"
- "compare this to similar tools in the space"
- "what patterns are we borrowing vs inventing?"
- "does this follow the mental model users expect?"

---

## 4. State Coverage

Prompts to try:

- "what states are missing from this wireframe?"
- "show me empty, loading, and error states"
- "what happens when there's no data?"
- "what does the first-run experience look like?"

---
